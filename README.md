# SimVault

<img src="assets/icon/SimVault_logo.png" width="80" alt="SimVault Logo" />

一款用于管理多张 SIM 卡和手机号码的移动应用。

## 功能

### 号码管理

<img src="screenshots/numbers.png" width="200" alt="号码管理" />

- 添加和管理多个手机号码
- 支持选择运营商（含自定义下拉选择）
- 记录号码的保号日期，追踪有效期
- 设置到期提醒，避免号码因欠费或未使用而被注销

### 发现

<img src="screenshots/discover.png" width="200" alt="发现" />

- 查看各运营商的详细介绍

### 我的

<img src="screenshots/profile.png" width="200" alt="我的" />

- 提醒设置：管理号码到期通知偏好
- 帮助与反馈：查看使用说明、提交使用反馈
- 关于：查看应用信息
